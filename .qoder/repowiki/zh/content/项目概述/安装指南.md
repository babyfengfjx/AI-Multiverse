## 简介

AI Multiverse Chat 是一个功能强大的 Chrome 扩展程序，允许用户同时向多个 AI 聊天平台发送消息。该扩展支持以下 AI 平台：
- Gemini (gemini.google.com)
- Grok (grok.com)
- Kimi (kimi.moonshot.cn)
- DeepSeek (chat.deepseek.com)
- ChatGPT (chatgpt.com)
- 通义千问 (Qwen) (chat.qwen.ai)
- 腾讯元宝 (Yuanbao) (yuanbao.tencent.com)

该扩展采用轻量级设计，使用原生 JavaScript、HTML 和 CSS 构建，无需复杂的构建工具。

## 系统要求

### 硬件要求
- **处理器**: 至少 1 GHz 双核处理器
- **内存**: 至少 512 MB RAM
- **存储空间**: 至少 10 MB 可用磁盘空间

### 软件要求
- **操作系统**: Windows 10/11、macOS 10.15+、Linux
- **浏览器**: Google Chrome 90+
- **网络连接**: 稳定的互联网连接
- **权限**: 需要访问扩展程序页面和相关网站

## Chrome 扩展安装步骤

### 步骤 1：打开 Chrome 扩展页面

1. **启动 Google Chrome 浏览器**
   - 确保使用最新版本的 Chrome
   - 关闭所有不必要的标签页以释放内存

2. **导航到扩展页面**
   - 在地址栏输入 `chrome://extensions`
   - 按下 Enter 键
   - 或者从菜单中选择"更多工具" → "扩展程序"

### 步骤 2：启用开发者模式

1. **找到右上角的开关**
   - 在扩展页面右上角找到"开发者模式"开关
   - 开关显示为"关闭"状态

2. **启用开发者模式**
   - 点击开关将其切换到"开启"状态
   - 页面会提示开发者模式已启用
   - 确认出现"开发者模式已启用"的提示信息

### 步骤 3：加载扩展程序

1. **点击"加载已解压的扩展程序"**
   - 在页面左侧找到"加载已解压的扩展程序"按钮
   - 点击该按钮

2. **选择扩展文件夹**
   - 导航到包含扩展文件的文件夹位置
   - 选择包含 `manifest.json` 文件的根目录
   - 点击"选择文件夹"或"打开"

3. **等待加载完成**
   - Chrome 会显示扩展加载进度
   - 加载完成后会出现扩展图标和名称

### 步骤 4：固定扩展程序

1. **找到扩展图标**
   - 在 Chrome 工具栏右侧找到扩展图标
   - 图标显示为 AI Multiverse Chat 的标志

2. **固定扩展**
   - 点击工具栏右侧的"拼图"图标
   - 在扩展列表中找到 AI Multiverse Chat
   - 点击"固定"图标（图钉形状）
   - 扩展现在会固定在工具栏上

### 步骤 5：验证安装

1. **检查扩展状态**
   - 确认扩展图标显示在工具栏上
   - 点击图标验证扩展正常工作

2. **查看扩展详情**
   - 在扩展页面中找到 AI Multiverse Chat
   - 点击"详细信息"查看扩展信息
   - 确认权限和版本信息正确

## 验证安装

### 方法一：通过扩展图标验证

1. **打开 Chrome 工具栏**
   - 查找 AI Multiverse Chat 图标
   - 图标应该显示在工具栏右侧

2. **点击扩展图标**
   - 点击图标打开扩展界面
   - 应该看到弹出窗口或侧边栏界面

### 方法二：通过扩展页面验证

1. **进入扩展管理页面**
   - 访问 `chrome://extensions`
   - 确认 AI Multiverse Chat 显示在扩展列表中

2. **检查扩展状态**
   - 确认扩展处于"启用"状态
   - 查看扩展的详细信息和权限

### 方法三：通过开发者工具验证

1. **打开开发者工具**
   - 按 F12 或右键选择"检查元素"
   - 切换到"控制台"选项卡

2. **检查扩展加载**
   - 在控制台中输入 `chrome.runtime.id`
   - 如果返回扩展 ID，说明安装成功

## 初始配置

### 配置 AI 平台选择

1. **打开扩展界面**
   - 点击扩展图标或打开侧边栏
   - 在弹出窗口中可以看到 AI 平台列表

2. **选择默认平台**
   - 默认情况下所有平台都已勾选
   - 根据需要取消不需要的平台
   - 建议保留常用的 3-4 个平台

### 设置快捷键

1. **自定义快捷键**
   - 在扩展页面中找到"键盘快捷键"
   - 点击"配置键盘快捷键"
   - 设置自定义快捷键组合

2. **推荐快捷键**
   - 默认快捷键：Alt+Shift+S
   - 可根据个人习惯调整

### 配置主题和语言

1. **切换主题**
   - 在侧边栏界面中找到主题切换按钮
   - 选择浅色或深色主题

2. **设置语言**
   - 点击语言切换按钮
   - 选择英语或中文界面

## 常见问题解决

### 问题 1：扩展无法加载

**症状**: 点击"加载已解压的扩展程序"后无反应

**解决方案**:
1. **检查文件完整性**
   - 确认所有文件都已下载完整
   - 特别检查 `manifest.json` 文件是否存在

2. **重新下载扩展**
   - 从官方源重新下载扩展包
   - 解压缩到新的文件夹

3. **检查文件权限**
   - 确保文件夹具有读取权限
   - 避免使用受保护的系统文件夹

### 问题 2：扩展显示为"未启用"

**症状**: 扩展出现在列表中但显示为禁用状态

**解决方案**:
1. **重新启用开发者模式**
   - 关闭并重新打开开发者模式
   - 重新加载扩展

2. **检查扩展权限**
   - 确认扩展所需的权限已授予
   - 检查浏览器的安全设置

### 问题 3：扩展图标不显示

**症状**: 扩展安装成功但工具栏中看不到图标

**解决方案**:
1. **固定扩展**
   - 在扩展页面中点击"固定"按钮
   - 确认扩展图标显示在工具栏

2. **重启浏览器**
   - 完全关闭 Chrome 后重新启动
   - 再次检查工具栏

### 问题 4：扩展无法访问某些 AI 网站

**症状**: 发送消息时某些平台无响应

**解决方案**:
1. **检查网络连接**
   - 确保网络连接稳定
   - 尝试访问目标网站确认可访问性

2. **清除浏览器缓存**
   - 清除 Chrome 缓存和 Cookie
   - 重新登录 AI 平台账户

3. **检查网站兼容性**
   - 确认目标网站仍在运营
   - 检查是否有网站维护通知

### 问题 5：扩展运行缓慢

**症状**: 扩展响应慢或卡顿

**解决方案**:
1. **关闭不必要的扩展**
   - 减少同时运行的扩展数量
   - 关闭不使用的扩展

2. **清理浏览器数据**
   - 清理浏览历史和缓存
   - 重启浏览器

3. **更新扩展**
   - 检查是否有扩展更新
   - 更新到最新版本

## 故障排除

### 系统兼容性问题

**Windows 用户**:
- 确保使用管理员权限运行 Chrome
- 检查 Windows Defender 防病毒软件设置
- 避免使用企业版 Chrome 策略限制

**macOS 用户**:
- 确认 macOS 版本满足要求
- 检查 Gatekeeper 安全设置
- 避免使用受限的用户账户

**Linux 用户**:
- 确认系统满足最低要求
- 检查桌面环境兼容性
- 确保有适当的文件权限

### 权限相关问题

1. **检查扩展权限**
   - 在扩展页面查看权限列表
   - 确认所有必需权限都已授予

2. **处理权限拒绝**
   - 重新安装扩展以重置权限
   - 清除浏览器数据后重新尝试

3. **安全软件干扰**
   - 暂时禁用安全软件测试
   - 将扩展添加到白名单

### 性能优化建议

1. **监控资源使用**
   - 使用 Chrome 任务管理器监控扩展资源
   - 定期重启浏览器释放内存

2. **优化扩展设置**
   - 减少同时运行的 AI 平台数量
   - 调整扩展的自动刷新频率

3. **定期维护**
   - 定期更新扩展到最新版本
   - 清理扩展的本地存储数据

## 卸载指南

### 完全卸载步骤

1. **禁用扩展**
   - 在扩展页面中点击"禁用"按钮
   - 确认禁用操作

2. **移除扩展**
   - 点击"移除扩展"按钮
   - 确认删除操作

3. **清理残留文件**
   - 删除扩展文件夹
   - 清理浏览器缓存

4. **重启浏览器**
   - 完全关闭 Chrome
   - 重新启动浏览器

### 清理浏览器数据

1. **清除扩展数据**
   - 访问 `chrome://settings/clearBrowserData`
   - 选择"扩展程序数据"
   - 点击"清除数据"

2. **重置扩展设置**
   - 在扩展页面中找到"重置扩展设置"
   - 确认重置操作
